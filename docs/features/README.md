# Features

Documentacion de las features implementadas en GitLab MR Board.

## Indice

| Feature | Descripcion |
|---|---|
| [Tablero Kanban](tablero-kanban.md) | Vista de MRs organizada en columnas por estado de mergeabilidad |
| [Clasificacion de mergeabilidad](clasificacion-mergeabilidad.md) | Logica que determina el estado de cada MR |
| [Multi-proyecto](multi-proyecto.md) | Soporte para multiples repositorios de GitLab en un solo tablero |
| [Busqueda y filtrado](busqueda-filtrado.md) | Filtrado en tiempo real de MRs por texto |
| [Auto-polling y cache](auto-polling-cache.md) | Refresco automatico de datos y cache del backend |
| [Blocker badges](blocker-badges.md) | Indicadores visuales de CI, threads y aprobaciones |
| [Responsable dinamico](responsable-dinamico.md) | Muestra quien es responsable de actuar sobre cada MR |
| [Rate limiting](rate-limiting.md) | Control de concurrencia en llamadas a la API de GitLab |
